# Configuration
Configuration is done via YAML file that needs to be in the `/config` folder.

```yaml
metrics:
  - name: "queue-size"
    resourceType: "Microsoft.ServiceBus"
  - name: "api-requests-count"
    resourceType: "Microsoft.WebApps"
```
